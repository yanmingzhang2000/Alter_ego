# Alter Ego业务背景

# 一、精简版完整业务背景汇总（适配 MVP 看板 \+ 配套 Agent）

## 1\. 组织架构体系

1. 层级：事业部 \> 工作室，单个事业部下包含多个业务工作室 \+ 事业部专属中台；全公司另有独立中台，归属 3MC 管理部独立核算盈亏。

2. 标准业务线：CMC、DMC、ECP、直播带货；各工作室可自主新增业务线，业务线长期固定。

3. 中台两类划分核算规则 

    - 事业部中台：归属对应事业部，对内支撑本事业部全部工作室，也可独立接单，成本利润计入所属事业部；

    - 独立中台（3MC）：单独部门，独立核算全部成本、利润，不归属任意事业部。

4. 人员规则 

    - 员工固定归属单一工作室 / 中台；支持跨工作室人力支援，所有跨团队项目单独提单区分核算；

    - 业务职能固定配置： DMC：BD 销售 \+ CM 客户经理 \+ AO 优化师 CMC：BD 销售 \+ CM 客户经理 \+ PM 项目经理 \+ KOL 运营

5. 客户 \& 订单核算规则 

    - 一个客户仅归属单一工作室；同一客户多项服务拆分为多张独立订单；多团队联合服务同一客户，通过多订单拆分营收、成本，不做一单内分摊。

## 2\. 收入、成本、利润口径（负责人核心关注）

### 收入

需同时展示三类口径，全部拆分新客户 / 老客户复购增购收入：签约额、回款额、财务交付确认收入。

### 成本 5 大类全部纳入核算

1. 直接人力成本（薪资、社保、绩效提成）

2. 广告投放采买成本

3. 第三方外包交付成本

4. 场地 / 工具 / 平台公共固定成本

5. 市场渠道获客费用

### 三层利润全部输出

项目毛利、工作室毛利、事业部扣除总部分摊后的净利；独立 3MC 中台单独计算自身毛利净利。

## 3\. 核心经营指标（市场负责人重点盯盘）

1. 人均绩效（多口径并行展示） 分子：人均营收、人均毛利、人均净利 分母：工作室总在职人数、当期实际交付在岗人数、仅 BD 销售岗人数

2. Margin 毛利率多层级展示：单项目毛利率、工作室综合毛利率、单客户毛利率

3. 客户全生命周期指标 分层：大客户、中小客户、流失客户分层查看利润 / 成本； 数量：新增客户、活跃客户、流失客户； 价值：单客服务成本、单客毛利、客户 LTV

## 4\. 底层数据现状

数据源 4 套带独立数据库：CRM 销售、BPMS 交付记账、TFG 计费绩效、Galaxy 财务； 痛点：系统相互割裂，无统一数据中间层，跨系统对账成本高；无统一经营分析基建。

## 5\. Agent 规划

暂无明确使用场景，MVP 阶段先落地看板，后续基于负责人日常经营分析需求迭代设计 Agent 能力。

## 6\. 特殊场景说明

跨工作室人力支援场景发生频次低，MVP 方案采用轻量化处理逻辑，不做复杂内部结算模型。

# 二、针对「跨工作室支援少」给你的 MVP 轻量化方案建议

1. 核算简化（优先满足 90% 常规业务，少量跨支援人工备注溯源，不复杂分摊） 

    - 人力成本默认归属员工所属工作室；

    - 跨工作室支援订单单独打标签，看板增加筛选维度，只做数据展示，MVP 不做双向内部利润冲抵结算；

    - 月度经营报表中单独汇总跨支援订单总规模，供负责人定性查看，不改动核心毛利计算逻辑。

2. 中台分摊 MVP 简化方案 

    - 事业部中台对内支撑成本：按各工作室当期营收占比分摊至对应工作室；中台自有独立订单单独核算，收入成本不参与分摊；

    - 3MC 独立中台完全独立视图，不向各事业部分摊成本，单独作为一个经营单元看板。

3. 公共固定成本统一分摊规则（MVP 定死单一标准，减少复杂度） 统一按当期营收占比分摊至事业部、工作室、业务线，后期版本再支持切换人头 / 工时分摊方式。

4. 阈值预警 MVP 简化 毛利率预警先配置通用统一阈值，支持负责人手动修改，不做多层级差异化复杂阈值配置。

5. 客户分层 MVP 极简标准 以年度签约金额划分大客户 / 中小客户；连续 90 天无新订单判定流失客户，标准固化，后续迭代开放自定义。

6. 数据口径优先级（解决多系统对账冲突） 金额类指标以财务 Galaxy 系统为准；业务订单、人员、客户数据以 CRM\+BPMS 为准，系统差异在看板增加差异提示标签，MVP 不做自动对账修正。

7. 数据更新频次 MVP 标准 核心指标 T\+1 日更新；月度完整经营快照固定每月 5 号自动生成，满足负责人日常看盘 \+ 月度复盘需求。

# 三、MVP 落地边界（砍掉复杂非刚需能力，快速交付可用看板）

1. 暂不开发复杂 AI Agent 交互能力，仅内置基础指标自动预警、月度经营快照自动导出；

2. 跨工作室人力调拨内部结算、多维度成本分摊切换、自定义客户分层规则放到二期迭代；

3. 权限仅两层：事业部 / 3MC 总负责人全视图、工作室负责人仅本团队数据，暂不细分岗位精细化权限；

4. 不支持复杂模拟测算（人力增减、调价利润预测），该能力并入二期 Agent 开发。
